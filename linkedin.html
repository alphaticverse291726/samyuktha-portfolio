How to add a LinkedIn post

  1. Go to your LinkedIn post → click the three dots (•••) → "Copy link to post"
  2. Open linkedin.html in a code editor
  3. Copy one of the .post-card blocks below
  4. Paste the post link in href="...", and fill in the title, excerpt, date, and stats
  5. Save the file and push to GitHub — it appears instantly on your site
